# 🚀 快速开始

本指南将帮助您快速开始使用 PixivFlow WebUI Frontend。

## 📋 前置要求

- Node.js 18+ 和 npm
- 后端服务器正在运行（参见主项目 README）

## ⚡ 快速安装

### 1. 安装依赖

```bash
cd webui-frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 打开浏览器

访问 `http://localhost:5173`

## 📋 开始之前

### 什么是 PixivFlow？

PixivFlow 是一个**自动化下载工具**，专门用于从 Pixiv（日本最大的插画分享网站）批量下载你喜欢的作品。

**主要特点**：
- 🤖 **自动化**：设置一次，自动运行，无需人工干预
- 🎯 **精准筛选**：按标签、收藏数、日期等条件筛选作品
- 💾 **智能去重**：自动跳过已下载的作品，避免重复
- 🔄 **稳定可靠**：自动重试、断点续传、错误恢复
- 🚀 **完全独立**：后端和脚本完全独立于前端，可在无前端环境下完美运行

### 环境要求

| 项目 | 要求 | 说明 |
|------|------|------|
| **Node.js** | 18.0.0 或更高 | JavaScript 运行环境 |
| **npm** | 9.0.0 或更高 | Node.js 的包管理器 |
| **Pixiv 账号** | 有效账号 | 用于登录认证 |
| **Python** | 3.9+（可选） | 用于登录获取 refresh token |
| **gppt** | `pip install gppt`（可选） | Python 包，用于 Pixiv 登录认证 |

**快速检查环境**：
```bash
node --version    # 应该显示 v18.0.0 或更高
npm --version     # 应该显示 9.0.0 或更高
```
